Welcome to Flask Templates!

This page is being served from an HTML file in the templates/ folder.

What's happening?

When you visit this URL, Flask:

  1. Receives your browser's request
  2. Matches the URL to the @app.route('/')
  3. Calls the home() function
  4. Loads templates/index.html
  5. Sends this HTML back to your browser

Why Use Templates?

  • Separation of Concerns: Python handles logic, HTML handles presentation
  • Easier to Maintain: Edit HTML without touching Python code
  • Reusability: Same template can be used with different data
  • Jinja2 Power: Add Python-like logic directly in HTML

Features of Flask

  • Lightweight and Minimalistic framework
  • Built-in development server and debugger
  • Supports secure cookies for client-side sessions
  • Easy integration with databases like SQLite, MySQL, and PostgreSQL
  • Flexible routing system for building web applications

How to Run This Project

  1. Make sure you have Python installed
  2. Create a virtual environment: python -m venv venv
  3. Activate the environment: source venv/bin/activate (Linux/Mac) or venv\Scripts\activate (Windows)
  4. Install Flask: pip install flask
  5. Run the app: python app.py
  6. Open your browser and visit http://localhost:5000

Tips for Beginners

  • Keep your templates in the templates/ folder
  • Use url_for() to generate links dynamically
  • Use Jinja2 loops and conditionals to avoid repetitive HTML
  • Keep CSS in static/ folder for better organization
